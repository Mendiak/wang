# W.A. Wang

A minimal portfolio for screenwriter and storyteller W.A. Wang.

The site uses a white background, near-black typography, an animated SVG wordmark, understated navigation, and restored Bio, Projects, and Awards sections.

## Core Files

* `index.html` - Semantic structure, hero, section content, contact links, and footer credit.
* `style.css` - Local display font, monochrome layout, responsive sections, and SVG text animation.
* `script.js` - Subtle animated SVG turbulence for the wordmark.
