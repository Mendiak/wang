# W.A. Wang — Screenwriter & Storyteller

A premium minimalist portfolio website for screenwriter and artist W.A. Wang. 

Designed with a luxury editorial aesthetic inspired by independent cinema and contemporary art portfolios.

## Key Features
* **Luxury Editorial Aesthetic:** High-end typography, generous whitespace, and pure monochrome layout.
* **Single-Viewport Layout:** No-scroll presentation on desktop and tablet, utilizing high-performance CSS and JS crossfades.
* **Mobile-Responsive Fallback:** Automatically switches to an elegant vertically stacked scrolling feed on mobile viewports.
* **Cinematic Micro-interactions:** Buttery 60fps linear interpolation (LERP) mouse physics, circular mask-depth parallax, and a quiet SVG text morphing filter.
* **Ultra-Lightweight & Clean:** Pure HTML, CSS, and Vanilla JS with zero external dependencies. Fully optimized for GitHub Pages.

## Core Files
* `index.html` — Semantic markup, accessibility features, and screenplay catalog details.
* `style.css` — Custom layout systems, editorial typography scales, and responsive transitions.
* `script.js` — Cinematic loader, view routing, magnetic mechanics, and hover animations.
